Available Courses

Calculus

learn to analyze and understand how quantities change and relate to each other through calculus.

Communications network

learn how data is transmitted, routed, and managed across interconnected systems in a communications network.

Digital logic

learn how to design and analyze digital circuits and systems using logic gates and binary operations.

Electric Circuits

learn how to analyze and design electrical circuits, including understanding components like resistors, capacitors, inductors, and how they work together to control electrical current and voltage.

Linear Algebra

learn how to analyze and solve systems of linear equations, work with vectors and matrices, understand vector spaces, and apply concepts like eigenvalues and eigenvectors to various mathematical and real-world problems.

Probability and Statistics

learn how to analyze data, understand likelihoods and chances of events, interpret statistical measures like mean, median, and standard deviation, and apply probability models to make informed decisions and predictions.

Physics for Engineers

learn fundamental principles such as mechanics (forces, motion, energy, momentum), thermodynamics, electromagnetism, and waves.

Java programming

teaches the fundamentals of Java language, including syntax, data types, control structures, object-oriented programming concepts, and basic application development.
